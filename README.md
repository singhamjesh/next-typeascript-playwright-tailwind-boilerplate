# Next.js + Typescript + Playwright + TailwindCSS Boilerplate

### Development Guidelines

#### Pre-required

- Node.js >= 18.15.0
- NPM >= 9.5.0

#### Technology stack

- Typescript
- Next.js
- TailwindCSS
- Playwright
- Storybook
- Redux
- Redux-persist
- Socket.io

#### To run the webapp

```
Clone project from github

npm install

npm run dev

```
